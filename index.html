Tracking Spending by Scraping Mint.com

Grocery Spending Tracker


My Family had a problem, we were spending $2,000 per month on groceries! One month we made 62 trips grocery stores. This is crazy, especially when we have a Costco membership

That's where this micro app comes in, I added this the Grocery Tracker to everyone's iphone homescreen. Now everyone in the family can check how much we are spending by tapping the icon on there phone, no logging in required!

This app uses the python library mintapi to scrape mint.com. The retrieved spending ammount is then uploaded to firebase where it is passed to an AgularJS app and displayed at the above link.


 
#Python Code for Scaping Mint.com
import sys
import mintapi
from firebase import firebase
from datetime import datetime
import time

#Grab Credentials
username = os.environ.get('USERNAME')
password = os.environ.get('PASSWORD')

#open text file for logging
loop_num = 1
log = open('mintUpdateLog.txt', 'a')

#image changes on each update
imgArray = ['img/food3.jpg','img/food2.jpg',
    'img/food1.jpg','img/fruit1.gif']
imgIdx = 0

#Scape mint every 30 minutes unless time is between 12am - 6am
while True:
    if loop_num > 15:
        break

    # Get Grocery Spending
    mint = mintapi.Mint(username, password)
    budgets = mint.get_budgets()

    groceries_name = budgets['spend'][5]['cat']['name']
    groceries_spent = budgets['spend'][5]['amt']

    #record time retrieved
    timenow = time.time()*1000

    #send Data to Firebase -->
    fbURL = 'https://walterbudget.firebaseio.com/data'
    myfirebase = firebase.FirebaseApplication(fbURL, None)

    #only storing one value delete previous data point (current expenditures)
    myfirebase.delete('/', 'data')

    #Add new spending amount
    myfirebase.post('/data', {groceries_name : groceries_spent, 'time': timenow,
        'url': imgArray[imgIdx]})

    log.write(str(groceries_name) + " " + str(groceries_spent) + " " +
        str(timenow)+ "\n")

    #choose the next img in line
    imgIdx = (imgIdx+1) % len(imgArray)

    currentHr = datetime.now().hour
    if currentHr < 6:
        time.sleep(18000)
    else:
        time.sleep(2340)

    loop_num+=1
    print loop_num
    print currentHr

log.close()
